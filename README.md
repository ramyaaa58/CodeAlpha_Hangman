# CodeAlpha_Hangman
This project is a simple and interactive Hangman game developed using Python as part of my internship. The game allows users to guess a hidden word letter by letter, with a limited number of attempts.
🛠️ Tech Stack:
Language: Python Concepts Used:
Loops, Conditionals, Lists, Strings, Random Module.
🚀Features:
Multiple difficulty levels (Easy, Medium, Hard)
Random word selection
Limited attempts for guessing
User-friendly console interaction
Step-by-step game progress display
📚What I Learned:
Implementing game logic using Python
Improving problem-solving and logical thinking skills
Handling user input and validations
Writing structured and readable code
▶️How to Run:
Install Python on your system
Save your file as hangman.py
Run the command:python hangman.py
The game will start and you can play 🎮
